Sports Ranking Widget for Sports Blogs Content by Blitzcorner.com

== Description ==

BlitzCorner Rankings allows users to pick and rank players, teams and more in a series of fun ranking categories.

== Installation ==

1. Unzip sports_ranking_widget.zip into /sports_ranking_widget folder.
2. /sports_ranking_widget folder should contain 3 files (sports_ranking_widget.php, readme.txt and screenshot-3.jpg)
3. Upload /sports_ranking_widget folder to your /wp-content/plugins/ directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Add the widget to your sidebar (Navigate to Appearance menu > Widgets and drag the widget to the prefered place on your sidebar)

== Frequently Asked Questions ==

= Can I limit Rankings to a specific sport? =

Yes! Check the box for the sport/s you'd like and get rankings categories specific in those sports. 
Note that some of the rankings combine athletes from different sports. Those are rankings are listed as "Other"

= How do I make the widget's design most compatible with my site's look? =

You can set the colors of the widget text and background to match your site's color's best. 

= I have an idea for a new Ranking category. How can I bring it to BlitzCorner's attention? =

Contact BlitzCorner Network at info@blitzcorner.com and your suggestion will be reviewed.
